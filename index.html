    Cómo Configurar tu Clave de API de Google

    Para que el asistente de Gemini funcione de forma segura y sin interrupciones, es crucial que restrinjas tu clave de API para que solo pueda ser usada desde tu página web. Sigue estos pasos:

    1. Ve a Google AI Studio

    Abre tu navegador y ve a Google AI Studio e inicia sesión.

    2. Crea o Encuentra tu Clave de API

    En el menú de la izquierda, haz clic en "Get API key". Si ya tienes una, cópiala. Si no, haz clic en "Create API key in new project" y copia la nueva clave.

    3. ¡Importante! Restringe tu Clave de API

    Este es el paso más importante para evitar que Google la bloquee.

    1. Después de copiar tu clave, haz clic en el enlace azul que dice "Edit key" o ve a la Consola de Google Cloud y busca tu clave.
    2. Dentro de la configuración de la clave, busca la sección llamada "Application restrictions" (Restricciones de la aplicación).
    3. Selecciona la opción "HTTP referrers (web sites)" (Referencias HTTP (sitios web)).
    4. En la sección de "Website restrictions" (Restricciones de sitios web), haz clic en "ADD" (AÑADIR).
    5. En el campo que aparece, escribe la dirección de tu catálogo, asegurándote de incluir el `*` al final para que funcione en todas las subpáginas. Por ejemplo: `tecnomarin.github.io/TUCEL/*`
    6. Importante: También debes añadir la URL específica que te dio el error para que funcione en este entorno de desarrollo. Cópiala y pégala exactamente así: 1b4zzhwoccf...usercontent.goog (Nota: esta URL temporal puede cambiar, si vuelve a fallar, revisa el nuevo error y añade la nueva URL).
    7. Haz clic en "Done" (Listo) y luego en "Save" (Guardar). Puede que los cambios tarden unos minutos en aplicarse.
    4. Añade la Clave a tu Catálogo

    Asegúrate de que la clave que copiaste esté en la variable `apiKey` en el código del catálogo.
